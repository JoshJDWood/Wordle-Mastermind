Wordle Mastermind

Welcome to Wordle Mastermind

In this game you must guess the 5 letter word. You have unlimited guesses to do this. For each guess you give you will receive hints: a green box for a correct letter in the correct space and a yellow box for a correct letter in the incorrect space. Use these hints to help you decipher the target word.

Colour code Letters by right clicking on them. Do this to both letters in your guesses and in the keyboard. All letters in a completely incorrect guess will turn grey. All letters in a completely correct guess will turn green. To match this I suggest colour coding:

  • GREY for Incorrect letters
  • YELLOW for Pressent letter
  • GREEN for Pressent letters in the Correct Position
